## Week 1 - How Machines Think

### What I built this week
This week, I successfully completed my first machine learning project using a Kaggle notebook. 
I explored data trends by building custom data charts and trained a Linear Regression machine learning model to calculate continuous numerical patterns. Additionally,
I experimented with natural language processing by testing an AI sentiment model, uncovering how text sentiment is analyzed and discovering specific edge cases where the 
machine's predictive patterns fail.

### What surprised me
I was surprised by how easily advanced AI models can be confused by basic human communication styles like sarcasm and mixed languages (Roman Urdu). Seeing the model classify 
a clearly sarcastic, negative complaint as 99.98% positive proved to me that machines don't actually understand the emotional depth of context; they are simply matching 
mathematical data weights of individual words.

### One question I still have
