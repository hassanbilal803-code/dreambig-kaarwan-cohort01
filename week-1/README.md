Week 1 - How Machines Think

What i  built this week
            This week, I executed an end-to-end machine learning workflow inside a Kaggle notebook. I performed 
Exploratory Data Analysis(EDA) using data visualization tools to identify underlying trends, and built and trained a
Linear Regression model and decission tree to predict continuous numerical values. Furthermore, I evaluated a pre-trained 
Natural Language Processing (NLP) sentiment analysis model to study token classification and identify systematic edge cases
where algorithmic predictions 
fail.

  What surprised me
                      I was surprised by how easily advanced language architectures are misaligned by nuanced human
communication styles like sarcasm and code-switching (Roman Urdu / 'Minglish'). Observing the model output a 99.98% positive
sentiment confidence score for an explicitly sarcastic, negative complaint demonstrated that these systems do not understand emotional context; 
rather, they map mathematical weights to isolated words.

 One question I still have
                        How do machine learning engineers scale the collection, data curation, and annotation of high-quality localized datasets (like Roman Urdu or Minglish) to eliminate these baseline semantic errors in South Asian 
software deployment pipelines?
